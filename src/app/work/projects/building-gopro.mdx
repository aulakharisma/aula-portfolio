## Overview

Developed a construction project management system using Laravel and Vite, designed for efficiency, enhancing workflow and team productivity at Pengawasan Investasi Rutin EPC Departemen, PT. Petrokimia Gresik.

## Technologies Used

- **Laravel**: For backend development.
- **Vite**: For efficient frontend development and faster build times.
- **MySQL**: For managing relational databases and ensuring data integrity.
- **Bootstrap**: For responsive and modern UI design.
- **Figma**: For creating and managing design assets and prototypes.
- **CPanel**: For Deployment and Server Management.

## Challenges and Learnings

One of the main challenges was integrating complex project workflows into a structured yet flexible system that could accommodate real-time updates and collaboration

## Outcome

The system is now actively used to streamline project tracking and management, leading to improved workflow transparency, better resource allocation, and enhanced collaboration between teams.
